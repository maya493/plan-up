عن Plan UP

موقع يساعد الطلاب على تنظيم المهام والمواعيد والامتحانات بطريقة بسيطة وعملية.

HTML • CSS • JavaScript

فكرة المشروع

Plan UP عبارة عن مخطط للدراسة: بتضيفي مهامك، بتحددي موعد وأولوية، وبتتابعي نسبة الإنجاز، وكمان بتضيفي الامتحانات وبيحسبلك كم يوم باقي.

المميزات

• تسجيل دخول / إنشاء حساب
• إضافة / تعديل / حذف المهام
• أولويات + حالة المهمة
• خطة أسبوعية حسب الموعد
• صفحة امتحانات مع عدّاد الأيام
• لكل مستخدم بيانات منفصلة (LocalStorage)

كيف يتم حفظ البيانات؟

يتم حفظ البيانات داخل المتصفح باستخدام LocalStorage لكل مستخدم بشكل منفصل. المشروع Frontend فقط (بدون قاعدة بيانات حقيقية) لكنه مناسب جدًا لمادة الويب.
